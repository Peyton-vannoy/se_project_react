# WTWR (What to Wear)

## Overview

WTWR is a web app that allows users to upload images of their clothing items and receive recommendations on what to wear based on the weather and their wardrobe.

## Features

- User Authentication
- Clothing Item Upload
- Weather-Based Recommendations
- User Profile

## Technologies Used

- React
- JavaScript
- HTML
- CSS

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server

frontend repository: https://github.com/Peyton-vannoy/se_project_react
